# oneliner.install

## Linux / MacOS

Suppose you want to install a project hosted at <http://github.com/horta/almosthere>.

```bash
bash -c "$(curl -fsSL https://git.io/Jz7Oa)" -s horta/almosthere
```

## Fetching version


```bash
bash -c "$(curl -fsSL https://bit.ly/36UxPPz)" -s horta/almosthere
```
